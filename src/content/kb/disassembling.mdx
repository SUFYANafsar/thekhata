				<div class="et_pb_text_inner"><p><strong>Add Disassembling</strong><br />In the Manufacturing menu, select Disassembling.</p>
<ol>
<li>Click <strong>“+Add Disassembling”</strong></li>
<li>Click <strong>Type to search assembly product</strong> and select the product you’re creating a Disassembling for.</li>
<li>Click <strong>Save and Continue Edit</strong> to save the Disassembling as a draft, or click <strong>Save and Approve</strong> to approve or click <strong>Save and Pending</strong> to mark the Disassembling as pending and enter a new Disassembling or click <strong>Save and New</strong> to record a new Disassembling and save the previous one or click <strong>Save and Close</strong> to close the Disassembling and save the Disassembling in draft status.</li>
</ol>
<p><strong>Disassembling fields and description</strong></p>
<ul>
<li><strong>Product to Disassemble:</strong> Enter the name of the product you would like to create a Disassembling for.</li>
<li><strong>Date, due date:</strong> This is set as default to the current date. You can change this also.</li>
<li><strong>Number:</strong> Use Spartan Accounts default numbering sequence or you can add your number sequence by clicking on number code and then a “+ “sign</li>
<li><strong>Reference:</strong> Choose a reference to suit your company. Later, you may want to search for your Disassembling using your chosen reference.</li>
<li><strong>Master Group:</strong> This field appears only if you have enabled Groups. Groups let you customize the Disassembling in many ways.</li>
<li><strong>Quantity to Disassemble:</strong> Enter the output quantity you want to discard for a product.</li>
<li><strong>Detail Group A:</strong> This field appears only if you have enabled Groups. Groups let you customize the Disassembling in many ways.</li>
<li><strong>Detail Group B:</strong> This field appears only if you have enabled Groups. Groups let you customize the Disassembling in many ways.</li>
</ul>
<p><strong>Disassembling line fields and Description</strong></p>
<ul>
<li><strong>Product:</strong> Select products if you have inventory products set up for your organization. Inventory products speed up data entry by automatically filling in the product field with information you’ve saved against your inventory products.</li>
<li><strong>Product Description:</strong> Enter text as you’d like to describe the products. You can include description lines in your product’s short description that will automatically display at line item.</li>
<li><strong>Warehouse:</strong> Select the warehouse in which the inventory will be managed. The default warehouse will be displayed in Disassembling form if you set the warehouse as default in the warehouse section. If there is only one warehouse, no dropdown will be available as all the inventory will be managed in that warehouse.</li>
<li><strong>Batch:</strong> This field will represent a batch number for the products</li>
<li><strong>Quantity:</strong> Enter the number of products.</li>
<li><strong>Cost:</strong> This will represent the average cost of a product managed in inventory.</li>
<li><strong>Amount:</strong> This is the calculated amount of a product.</li>
<li><strong>Consignment:</strong> Add a consignment number to record from against which consignment has been received. The consignment will display if you have enabled tracking the consignment from company settings.</li>
</ul>
<p><strong>Account Expenses (Optional):</strong></p>
<p>This is an optional field to record your expenses such as delivery charges or shipping charges.<br /><img data-lazyloaded="1" decoding="async" class="center" src="/wp-content/uploads/2021/08/Disassembling-Account-Expenses.jpg" alt="Account Expenses " style="display: block; margin-left: auto; margin-right: auto;" /><noscript><img decoding="async" class="center" src="/wp-content/uploads/2021/08/Disassembling-Account-Expenses.jpg" alt="Account Expenses " style="display: block; margin-left: auto; margin-right: auto;" /></noscript></p>
<p><strong>Manufacturing Setting</strong></p>
<p>Spartan Accounts allow you to manage settings for input and output quantity at assembling or disassembling. These settings can be done in company setting</p></div>
