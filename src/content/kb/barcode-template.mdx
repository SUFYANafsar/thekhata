				<div class="et_pb_text_inner"><p>A barcode essentially is a way to encode information in a visual pattern that a machine can read. The combination of black and white bars (elements) represents different text characters that follow a set algorithm for that barcode type.</p>
<p><strong>Add Barcode Template</strong></p>
<p>With Barcode Templates, you can create barcode reports using values from your records in Splendid Accounts. For example, create stickers with UPC barcodes for all of your inventory that can be scanned when picking items for an order. Any value in a field in Splendid Accounts can be used. A PDF file of the barcode(s) will be generated that you can then print as needed.</p>
<p>Here&#8217;s an example of a barcode label that was created to show the Product code:</p>
<p><img decoding="async" class="center" src="/wp-content/uploads/2021/08/Barcode.jpg" alt="Barcode" style="display: block; margin-left: auto; margin-right: auto;" /></p></div>
			</div><div class="et_pb_module et_pb_text et_pb_text_2  et_pb_text_align_left et_pb_bg_layout_light">
				
				
				
				
				<div class="et_pb_text_inner"><h2 style="text-align: left;">Create a Barcode Template</h2>
<p>Click the Add Barcode template in the top right. The Add Barcode Template screen appears:</p>
<p><img decoding="async" src="/wp-content/uploads/2021/09/Create-a-Barcode-Template-2.jpg" alt="Create a Barcode Template" style="display: block; margin-left: auto; margin-right: auto;" /></p>
<ol>
<li>Enter a name that describes the template name or other relevant information for you to identify the template (e.g., Order Number Barcode).</li>
<li>Barcode general, enter the Font size, width ,height, Quantity of the stickers that you will be printing on and No of Columns  (e.g. if printing on a 4&#8243; x 3&#8243; sticker sheet).</li>
<li>Barcode Margin, set the margin spacing of left, right, top, bottom margin spacing along with the size of Company name, Product name, Barcode Label, Product Price</li>
<li>Barcode Padding, set the padding spacing of left, right, top, bottom padding spacing along with the size of Company name, Product name, Barcode Label, Product Price</li>
<li>Comapny Name, set the font size and position of the company name in sticker</li>
<li>Product Name,set the font size and position of the product name in sticker</li>
<li>Barcode Label, set the font size and position of the barcode name in sticker</li>
<li>Product Price,set the font size and position of the price name in sticker</li>
<li>MRP, check the checkbox to display MRP in sticker</li>
</ol></div>
