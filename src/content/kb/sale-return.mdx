				<div class="et_pb_text_inner"><h2>Create a Return​</h2>
<h2>Overview​</h2>
<p>If you&#8217;re adding or editing the return, here&#8217;s some guidance on entering information into the fields.</p>
<ol>
<li>In the Sales menu, select return.</li>
<li>Click “<strong>+Add Sale </strong><strong>return</strong>”</li>
<li>Enter the name of the customer you&#8217;re creating the return If you use multi-currency, you can enter your return in the currency of the customer.</li>
<li>If the customer is not existing in the system you can directly add the customer right from the current screen by just writing the customer name and click enter, a form will be open for the additional customer you can save it and make a further transaction.</li>
<li>Click <strong>Type to search product </strong>and select the product, if it is not existing then you can enter the product directly the same as the customer name. Type the description of the product, then add the <b>Quantity and</b> <strong>Price</strong>.</li>
<li>You can add a line item <strong>discount</strong> and <strong>multiple taxes</strong> in a single product as per your needs.</li>
<li>You can also add <strong>Shipping Charges</strong> directly in the return at the bottom right corner which will record as shipping charges expense.</li>
<li>Click <strong>Save and Continue Edit to</strong> save the return as a draft, or click <strong>Save and Approve</strong> to approve or click <strong>Save and Pending</strong> to mark the return as pending and enter a new return or click <strong>Save and New</strong> to record a new return and save the previous one or click <strong>Save and Close</strong> to close the return and save the return in draft status.</li>
</ol>
<p><img data-lazyloaded="1" decoding="async" class="center-kb-image" src="/wp-content/uploads/2021/08/Create-a-Return.jpg" alt="Create a Return" style="display: block; margin-left: auto; margin-right: auto;" /><noscript><img decoding="async" class="center-kb-image" src="/wp-content/uploads/2021/08/Create-a-Return.jpg" alt="Create a Return" style="display: block; margin-left: auto; margin-right: auto;" /></noscript></p>
<p><strong><u>Header Fields:</u></strong></p>
<ul>
<li style="list-style-type: none;">
<ul>
<li><strong>Customer:</strong> Enter the name of the customer you would like to sell your products to. If you enter a name that&#8217;s not already a contact, Splendid Account adds it to your Customer list.</li>
<li><strong>Date:</strong> This is set as default to the current date. You can change this also.</li>
<li><strong>Number:</strong> Use Spartan Accounts default numbering sequence or you can add your number sequence by clicking on number code and then a “+ “sign</li>
<li><strong>Reference:</strong> Choose a reference to suit your company. Later, you may want to search for your return using your chosen reference.</li>
<li><strong>Master Group:</strong> This field appears only if you have enabled Groups. Groups let you customize the return in many ways.</li>
<li><strong>Currency:</strong> If you have a Business Edition plan with multi-currency, you can select a foreign currency customer that you&#8217;ve already added, you can edit the exchange rate if required. If you entered a default currency customer, it automatically displays.</li>
<li><strong>Exchange Rate:</strong> This field will display if you select a customer of such currency that is different from your company’s currency so that you have to provide an exchange rate for the particular customer.</li>
<li><strong>Order Booker: </strong>This field appears only if you have enabled. Let your customer know which order booker will go and collect the order. You can add multiple order bookers under the Sales Person menu.</li>
<li><strong>Sales Man: </strong>This field appears only if you have enabled. This will display a list of all the salesmen you have entered into the system.</li>
<li><strong>Delivery Person: </strong>This field appears only if you have enabled. Select the delivery person of your choice for a particular customer, also you can choose based on areas.</li>
<li><strong>Add Quick/Scan Product:</strong> You can also Scan your multiple products directly from the Barcode scanner or enter a product name or Product number through the Add Quick product box.</li>
</ul>
</li>
</ul>
<p><img data-lazyloaded="1" decoding="async" class="center-kb-image" src="/wp-content/uploads/2021/08/branches-header-feild.jpg" alt="Add Quick/Scan Product" style="display: block; margin-left: auto; margin-right: auto;" /><noscript><img decoding="async" class="center-kb-image" src="/wp-content/uploads/2021/08/branches-header-feild.jpg" alt="Add Quick/Scan Product" style="display: block; margin-left: auto; margin-right: auto;" /></noscript></p>
<p><strong><u>Line Item Fields:</u></strong></p>
<ul>
<li><strong>Product: </strong>Select products if you have inventory products set up for your organization. Inventory products speed up data entry by automatically filling in the product field with information you&#8217;ve saved against your inventory products.<br />If you enter a product that&#8217;s not already in inventory, Splendid Account adds it to your inventory.</li>
<li><strong>Product Description:</strong> Enter text as you&#8217;d like to describe the products. You can include description lines in your product’s short description that will automatically display at line item.</li>
<li><strong>Warehouse</strong>: Select the warehouse in which the inventory will be managed. The default warehouse will be displayed in return form if you set the warehouse as default in the warehouse section. If there is only one warehouse, no dropdown will be available as all the inventory will be managed in that warehouse.</li>
<li><strong>Quantity: </strong>Enter the quantity number of products you want to return.</li>
<li><strong>Price:</strong> Spartan Accounts display the default product price that you have set in the product form or also you can enter your price.</li>
<li><strong>Last Sale Price:</strong> This will display if you click in the price field, representing the last selling price of the product.</li>
<li><strong>Discount: </strong>Spartan Accounts has 3 different types of line item discount i.e.<br /><strong>(1. Discount% 2. Discount in Amount 3. Discounted price per unit)</strong> you can enter your discount as per supplier agreement.</li>
<li><strong>Tax: </strong>Splendid Account has an option to add multiple Tax on products. If you have set up the product tax in product form default tax will be display. Or you can add additional tax into it.</li>
<li><strong>Detail Group A:</strong> This field appears only if you have enabled Groups. Groups let you customize the return in many ways.</li>
<li><strong>Detail Group B: </strong>This field appears only if you have enabled Groups. Groups let you customize the return in many ways.</li>
<li><strong>Amount: </strong>This is the calculated amount customer should receive against a product.</li>
<li><strong>Consignment:</strong> Add a consignment number to record from against which consignment has been returned. The consignment will display if you have enabled tracking the consignment from company settings.</li>
</ul></div>
