				<div class="et_pb_text_inner"><h2>SMTP Settings</h2>
<p>Default Spartan Accounts mail settings are proceeded for sending emails from Sale Quotation, invoices but if you want to use your email address, you can set it at SMTP settings under company settings. Email address settings for outgoing emails can be set in the SMTP tab under company settings. If you allow your email address to send emails you to provide the following fields to work smoothly.<br /><img data-lazyloaded="1" decoding="async" class="center" src="/wp-content/uploads/2021/09/SMTP-Settings.jpg" alt="SMTP Settings" style="display: block; margin-left: auto; margin-right: auto;" /><noscript><img decoding="async" class="center" src="/wp-content/uploads/2021/09/SMTP-Settings.jpg" alt="SMTP Settings" style="display: block; margin-left: auto; margin-right: auto;" /></noscript><br /><strong>From Display Name: This is the display name for the email address you have chosen from which emails will send.</strong></p>
<p><strong>From Email Address: The email address is used to send emails from.</strong></p>
<p><strong>Reply to Email Address: The email address from which all the replies will be made.</strong></p>
<p><strong>Host: Choose the hostname correctly.</strong></p>
<p><strong>Port: Select the port number accordingly.</strong></p>
<p><strong>Enable SSL: If you want to enable SSL, check the checkbox</strong></p>
<p><strong>SMTP UserName: Enter the email address having the same host.</strong></p>
<p><strong>SMTP Password: Enter the password of the email entered in SMTP Username.</strong></p>
<p><strong>API Keys</strong></p>
<p>You can generate API Keys in Spartan Accounts.<br /><img data-lazyloaded="1" decoding="async" class="center" src="/wp-content/uploads/2021/09/API-Keys.jpg" alt="API Keys" style="display: block; margin-left: auto; margin-right: auto;" /><noscript><img decoding="async" class="center" src="/wp-content/uploads/2021/09/API-Keys.jpg" alt="API Keys" style="display: block; margin-left: auto; margin-right: auto;" /></noscript><br /><strong>Brand List</strong></p>
<p>You can add brands against a product under product details but before that, you have to add multiple brands in Spartan Accounts.<br /><img data-lazyloaded="1" decoding="async" class="center" src="/wp-content/uploads/2021/09/brand.jpg" alt="Brand" style="display: block; margin-left: auto; margin-right: auto;" /><noscript><img decoding="async" class="center" src="/wp-content/uploads/2021/09/brand.jpg" alt="Brand" style="display: block; margin-left: auto; margin-right: auto;" /></noscript><br /><strong>Add Brands</strong></p>
<p>You can add a brand in our system by visiting the setup menu and the select brand under the categories column.</p>
<p><strong>Edit Brands</strong></p>
<p>You can edit the brand name by going to the brand list and select the desired brand.</p>
<p><strong>Delete Brands</strong></p>
<p>If the brand is not associated with any transactions in the system then the brand can be deleted from the system.</p>
<p><strong>Print via PDF or Excel sheet</strong></p>
<p>Once the brand list has been added into the system, you can print the list of all the brands via PDF or Excel Sheet.</p></div>
